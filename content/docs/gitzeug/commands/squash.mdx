---
title: squash
description: Squashes the last n commits.
---
### Usage
```sh
gitzeug squash <NUMBER> <CMTMSG>
```
#### Options
 - `<NUMBER>` &nbsp;&nbsp;&nbsp;&nbsp;Number of commits to squash.
 - `<CMTMSG>` &nbsp;&nbsp;&nbsp;&nbsp;New commit message.

### Example
#### Squash the last 3 commits with a new commit message
```sh
gitzeug squash 3 "This is a squash of the last 3 commits"
```
