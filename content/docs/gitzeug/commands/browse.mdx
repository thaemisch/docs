---
title: browse
description: Shows the file tree for the git repo.
---
### Usage
```sh
gitzeug browse <URL>
```
#### Options
 - `<URL>` &nbsp;&nbsp;&nbsp;&nbsp;The URL of the git repo.

### Example
#### Browse the file tree of the gitzeug repository
```sh
gitzeug browse https://github.com/thaemisch/gitzeug.git
```
