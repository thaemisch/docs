---
title: dl
description: Downloads the specified directories and files from a git repo.
---
### Usage
```sh
gitzeug dl -u <URL> -o <PATH> -f <FILES>
```
#### Options
 - `-u`, `--url <URL>` &nbsp;&nbsp;&nbsp;&nbsp;The URL of the git repo.
 - `-o`, `--output <PATH>` &nbsp;&nbsp;&nbsp;&nbsp;The ouput/destination directory.
 - `-f`, `--files <FILES>` &nbsp;&nbsp;&nbsp;&nbsp;Directories and files to download.

### Example
#### Download the `src` directory and `Cargo.toml` file from the gitzeug repository
```sh
gitzeug dl -u https://github.com/thaemisch/gitzeug.git -o . -f src,Cargo.toml && ls -la
```
