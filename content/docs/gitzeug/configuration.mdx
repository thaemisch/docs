#### Disclaimer
Please note that configuration is purely optional and the file is not required for gitzeug to function.

### Initialization
To initialize the config file for gitzeug, run the following command:

```sh
gitzeug config
```

This will create a `gitzeug` directory in your default config directory (usually `~/.config`), containing the configuration file `config.toml`.
You can now open this file and edit it to customize gitzeug's behavior.
